== USER

=== Login API
/api/user/login

email, password를 입력받아 DB의 유저 정보와 비교

- 성공 : email, token 반환
- 실패 : 403 Forbidden

token은 이후 HTTP Header에 X-Auth-Token 키 값으로 token 값을 넣어서 요청하여 인가 확인

=== Register API
/api/user/register

- 성공 : 201 Created
- 실패 : 400 Bad Request
